# Plano de Testes de Software
 
| **Caso de Teste** 	| **CT-01 – Cadastrar usuário gestores de ONG** 	|
|:---:	|:---:	|
|	Requisito Associado 	| RF-001 - O sistema permitirá cadastro de usuários gestores de ONG. |
| Objetivo do Teste 	| Verificar se o usuário consegue se cadastrar na aplicação. |
| Passos 	| - Acessar o navegador; <br> - Informar o endereço do site; <br> - Clicar em "Sou uma ONG"; <br> - Preencher os campos obrigatórios (e-mail, nome, celular, área de atuação,senha, confirmação de senha); <br> - Aceitar os termos de uso; <br> - Clicar em "Registrar". |
|Critério de Êxito | - Uma mensagem informando que o cadastro foi realizado com sucesso devera aparecer e o usuario devera ser redirecionado para a pagina home com o login efetuado. |
